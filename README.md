# HackSC-2025-Echoes
Won Best Overall HackSC 2025: Echoes is an AI-powered speech-therapy companion built using AV-HuBERT (433h), MediaPipe, and PyTorch to assist individuals with autism and post-stroke speech impairments. It integrates lip-reading, emotion detection, and real-time clinician feedback, providing an accessible and responsive therapy support system.
